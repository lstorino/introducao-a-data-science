# introducao-a-data-science

Short description
Notebooks and exercises from an introductory Data Science course (Alura). Good learning artifacts and examples of common DS workflows.

Badges
- (language: Jupyter Notebook)

What’s included
- Sequential notebooks covering data exploration, cleaning, visualization, and basic modelling.
- Example datasets and exercise answers.

Quick start (recommended)
1. Open notebooks in Google Colab or locally:
   - In Colab: Upload the notebook or open via GitHub integration.
2. To run locally:
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   jupyter lab
